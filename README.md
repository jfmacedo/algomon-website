# 🎮 Algomon - Interactive Game Website

## 📌 Overview

Algomon is a multi-page interactive website developed as part of a **Client Side Programming** module at CCT College Dublin.

The project demonstrates core front-end development skills using **HTML, CSS, and JavaScript**, focusing on user interaction, UI design, and structured web development.

This version has been migrated from a private repository and is now being actively improved as part of my public portfolio.

---

## 🚀 Features

* Multi-page website (Home, About, Gallery, Battle Tips, Lore & Legends)
* Responsive navigation bar
* Interactive personality quiz
* Dynamic content rendering using JavaScript
* Custom fantasy-themed UI design
* Animated particle background for visual enhancement

---

## 🧠 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 💡 What I Learned

* Styling and layout design with CSS
* Implementing interactivity with JavaScript and event listeners
* DOM manipulation and user input handling
* Designing user-friendly interfaces with a consistent theme
